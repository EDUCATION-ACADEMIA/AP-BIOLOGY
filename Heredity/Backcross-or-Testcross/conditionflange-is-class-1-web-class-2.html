Condition:Flange is Class 1, Web Class 2;

(A) Thus, the section belongs to Class 1

(B) ** Thus, the section belongs to Class 2

(C) Thus, the section belongs to Class 3

(D) Thus, the section belongs to Class 4

EXPLANATIONS BELOW

Concept note-1: -Class 1 cross-sections are those which can form a plastic hinge with the rotation capacity required for plastic analysis without reduction of the resistance. Class 2 cross-sections are those which can develop their plastic moment resistance, but which have limited rotation capacity because of local buckling.

Concept note-2: -Cross-sections with class 3 webs and class 1 or 2 flanges may be classified as Class 2 cross-sections with an effective web ( Clause 6.2. 2.4 of EC3-1-1)

Concept note-3: -Limiting value of c/t for Class 1 in compression is 9 = 7.29. Actual c/t = 6.34 < limiting c/t (7.29) Therefore, the flanges are Class 1.

Concept note-4: -The rolled steel tee sections are classified into the following five series as per ISI: Indian Standard Normal Tee Bars ISNT. Indian Standard Wide flange Tee Bars ISHT. Indian Standard Long Legged Tee Bars ISST.
